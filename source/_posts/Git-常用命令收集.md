# Git常用命令收集

## git clone 
- git clone
- git clone -b branchname
    克隆特定分支
- git clone --depth depthlength
  克隆指定深度 
- ssh -T git@github.com
　测试ssh连接

## git checkout
- git checkout 
  用暂存区覆盖工作区，即回撤本地修改

- git checkout -b branchname
  创建并切换到分支
